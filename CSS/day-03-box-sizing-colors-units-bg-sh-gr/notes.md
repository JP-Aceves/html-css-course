## Day 03 - Box Sizing - Colors - Units - Backgrounds, Gradients and Shadows - Practical Exercise

### What I want to learn
I want to learn and complete these sections:
1. Box Sizing
2. Colors
3. Units
4. Backgrounds, gradients and shadows

### What I learned
I learned how the size of boxes is defined by multiple factors — not just width and height, but also padding, border, and how box-sizing handles all of them together.

I also explored different ways to define colors: hexadecimal, rgb, rgba, and color names, and how background and text colors need to work together visually.

On the units side, I now understand that measurements in CSS go beyond pixels. rem and em are relative units (1rem = 16px by default) that scale better across devices, while percentages are ideal for flexible layouts. I don't have every use case memorized yet, but I know the standard is rem for text and spacing, px for fixed details, and % for widths.

Finally, I practiced background images, gradients, and shadows — each with their own set of attributes like background-size, background-repeat, linear-gradient direction and colors, and box-shadow values for offset, blur, spread, and opacity.
